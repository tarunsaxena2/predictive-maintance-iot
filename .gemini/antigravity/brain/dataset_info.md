# Dataset Information

Dataset:
AI4I 2020 Predictive Maintenance Dataset

Target Column:
Machine failure

Important Features:
- Air temperature
- Process temperature
- Rotational speed
- Torque
- Tool wear

Task Type:
Binary Classification

Evaluation Metric:
Macro F1 Score

Dataset Size:
10000 records

Failure Types:
- TWF (Tool Wear Failure)
- HDF (Heat Dissipation Failure)
- PWF (Power Failure)
- OSF (Overstrain Failure)
- RNF (Random Failure)

Source:
UCI Machine Learning Repository
